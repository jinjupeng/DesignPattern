# 设计模式

　　① 01.单例（Singleton）模式

　　② 02.简单工厂（Simple Factory）模式

　　③ 03.工厂方法（Factory Method）模式

　　④ 04.抽象工厂（Abstract Factory）模式

　　⑤ 05.原型（Prototype）模式

　　⑥ 06.建造者（Builder）模式

### 结构型模式

　　① 07.适配器（Adapter）模式

　　② 08.桥接（Bridge）模式

　　③ 09.组合（Composite）模式

　　④ 10.装饰（Decorator）模式

　　⑤ 11.外观（Facade）模式

　　⑥ 12.享元（Flyweight）模式

　　⑦ 13.代理（Proxy）模式

### 行为型模式

　　② 15.观察者（Observer）模式

　　③ 16.访问者（Visitor）模式

　　④ 17.模板方法（Template Method）模式

　　⑤ 18.策略（Strategy）模式

　　⑥ 19.命令（Command）模式

　　⑦ 20.备忘录（Memento）模式

　　⑧ 21.迭代器（Iterator）模式

　　⑨ 22.中介者（Mediator）模式

　　⑩ 23.解释器（Interpreter）模式



## UML图

虚线箭头指向依赖；
实线箭头指向关联；
虚线三角指向接口；
实线三角指向父类；
空心菱形能分离而独立存在，是聚合；
实心菱形精密关联不可分，是组合；
